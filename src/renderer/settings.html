Confidentialité

Mxlw Browser est conçu pour protéger votre vie privée. Ces paramètres vous permettent de contrôler vos données.

Protection des données

Bloqueur de publicités
Bloque les publicités intrusives et les trackers pour une navigation plus rapide et sécurisée.
Protection contre le pistage
Empêche les sites web de vous suivre à travers différents sites.
Mode de navigation privée par défaut
Ouvre automatiquement tous les nouveaux onglets en mode navigation privée.

Cookies et données de site

Supprimer les cookies à la fermeture
Supprime automatiquement tous les cookies lorsque vous fermez le navigateur.

Sécurité

Ces paramètres affectent la sécurité de votre navigation. Il est recommandé de les laisser activés.

Protection web

Bloquer les sites dangereux
Empêche l'accès aux sites signalés comme malveillants ou de phishing.
Avertissement pour les sites non sécurisés (HTTP)
Affiche un avertissement lorsque vous visitez un site utilisant une connexion non chiffrée.
Vérifier les certificats SSL
Vérifie la validité des certificats de sécurité des sites web.

Contrôle des téléchargements

Demander où enregistrer les fichiers
Demander l'emplacement d'enregistrement pour chaque téléchargement.
Analyser les fichiers téléchargés
Vérifie les fichiers téléchargés pour détecter les logiciels malveillants.

À propos de Mxlw Browser LTS

Version 1.1.1 LTS-S-G

Mxlw Browser est un navigateur web moderne, rapide et sécurisé développé par Maxlware Development.

Basé sur Electron et Chromium, il combine la puissance de Chromium avec une interface utilisateur élégante et des fonctionnalités de confidentialité avancées.
